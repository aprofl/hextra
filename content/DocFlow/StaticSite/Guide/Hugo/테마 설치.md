## 설치

### ~~Hugo와 Node.js 설치~~
### Hyas와 Doks 템플릿을 사용하여 프로젝트 생성

```sh
npm create hyas@latest Aprofl_Hugo -- --template doks
cd DocFlow_Hugo
```

### 프로젝트의 의존성 설치

```sh
npm install
```

### 개발 서버 시작

```
hugo server
```
